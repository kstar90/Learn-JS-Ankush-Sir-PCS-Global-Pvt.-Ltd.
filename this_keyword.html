A) What is "this" keyword ? -> "this" keyword is mostly used in object-oriented programming languages like Java, JavaScript, C++, etc. B) what is the meaning of "this" ? -> "this" means it is current object or current instance. C) When we are inside any class and need to write "this" keyword then what is the meaning of writing "this"? -> That means we are indicating the object which is currently active. D) Why we use "this" keyword ? -> "this" keyword is used in the following cases: i) When the name of instance variables and local variables are same. ii) When we need the reference of an object. iii) When we need to do method chaining or constructor chaining. E) Code Example -> class Student { String name; Student(String name) { this.name = name; // yahan "this.name" ka matlab: class ka variable // aur "name" ka matlab: constructor ka parameter } void printName() { System.out.println(this.name); // yahan "this" current object ko refer karta hai } }
